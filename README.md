# -NPP-Nekoyama-Prefecture
猫山県及び猫山高速鉄道は、2024/8/10に創設された日本RP × 鉄道RPの世界です。県全体の世界観を大切にしながら、猫山高速鉄道などの鉄道イベントも楽しめます。まだ一周年ですが、Robloxグループのメンバーは280人を突破しました。現在は、日本RPの“しっかりした土台づくり”に力を入れており、その上で本格的な鉄道RPも展開できる世界を形作っています。まだ発展途中のプロジェクトだからこそ、参加してくれる皆さんの存在が大きな支えになります。Discord参加は開発のモチベーションを大きく上げてくれます！ Discord: https://discord.gg/ajzcN4YYBV
